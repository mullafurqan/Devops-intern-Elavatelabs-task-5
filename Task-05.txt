1. Install and Start Minikube
1.1 Download Minikube (Windows PowerShell Admin)
Invoke-WebRequest -Uri https://storage.googleapis.com/minikube/releases/latest/minikube-installer.exe -OutFile minikube.exe

1.2 Run installer
.\minikube.exe

1.3 Start the Kubernetes cluster
minikube start

1.4 Check Minikube status
minikube status

2. Install kubectl (if not installed)
2.1 Install using Chocolatey
choco install kubernetes-cli

2.2 Verify installation
kubectl version --client

3. Create Kubernetes Project Folder
3.1 Create a working directory
mkdir kubernetes-minikube
cd kubernetes-minikube

4. Create Deployment for Application
4.1 Create file: deployment.yaml

(Add this content)

apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 1
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:latest
        ports:
        - containerPort: 80

4.2 Apply deployment
kubectl apply -f deployment.yaml

4.3 Verify deployment & pods
kubectl get deployments
kubectl get pods

5. Create Service to Expose App
5.1 Create service.yaml

(Add this content)

apiVersion: v1
kind: Service
metadata:
  name: nginx-service
spec:
  type: NodePort
  selector:
    app: nginx
  ports:
    - protocol: TCP
      port: 80
      targetPort: 80
      nodePort: 30007

5.2 Apply service
kubectl apply -f service.yaml

5.3 Verify service
kubectl get svc
